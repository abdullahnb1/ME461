Set your wifi and usb settings in the pico and pc code.

Save max7219.py and tetris_pico_v3.py to pico as "max7219.py" and "main.py" respectively using thonny.

Unplug and plug pico when you want to run the game. Also make sure thonny is closed.

After you plug pico run tetris_pc_v3.py

Pieces are controlled with wasd and arrow keys. "P" pauses the game.
